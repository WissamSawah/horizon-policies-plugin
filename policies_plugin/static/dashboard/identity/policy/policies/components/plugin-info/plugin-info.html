Info

Table settings

Underneath the help/info button there is a table settings button, clicking on this button will open a menu allowing the user to change which fields are visible as well as how many policies are shown per page.


Search settings

The search-bar defaults to search in every field for the entered value. This can be changed by clicking on the magnifying glass button on the right of the searchbar, this will open up a panel beneath/underneath the search-bar with options to de-select fields to exclude them from the search.


Edit policy

To edit a policy using the form editor either click on a policies rule or right click on a policy and chose edit. This will open a form that will let the user edit and save a policy rule.


Edit policy using JSON format

To edit a policy using the form editor either click on a policies rule or right click on a policy and chose edit. This will open a form that will let the user edit and save a policy rule.


Downloading & Uploading files

The plugin supports downloading and uploading json files as a way to backup policies and restoring policies from a backup.

Downloading policies can be done by selecting policies using the checkboxes and clicking on the download button, this can also be done in the text editor. To download a single policy right click on a policy and hover over more and an option to download will appear.

Uploading policies can be done via the text editor, after clicking on the upload from file button a panel will appear where the user can chose a file to upload.

The contents of the file will appear in the text editor after being uploaded to let the user make any changes if necessary. The upload feature supports json files and will show an error if the user tries to upload another type of file.
